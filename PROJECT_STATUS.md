# MacAgent 项目状态报告

> 生成时间：2026-02-24

## 📊 项目概述

MacAgent 是一个本地 macOS AI Agent 助手，支持多种 LLM 提供商（DeepSeek、Ollama、LM Studio），具备自主执行任务的能力，能够调用系统级工具完成各种操作。

### 6.1 自适应停止策略 (v0.7.0 新增)

| 功能 | 状态 | 描述 |
|------|------|------|
| `AdaptiveStopPolicy` | ✅ 完成 | 智能停止策略，多条件触发 |
| `ProgressTracker` | ✅ 完成 | 进度追踪，检测任务是否有进展 |
| `LoopDetector` | ✅ 完成 | 循环检测，识别重复模式 |
| `CostTracker` | ✅ 完成 | 成本追踪（Token/时间） |
| `TaskComplexityAnalyzer` | ✅ 完成 | 任务复杂度分析，自适应迭代上限 |

**停止条件：**
```
AgentStopPolicy =
    MaxIterations      # 达到自适应最大迭代次数
  | MaxCost            # 达到 Token/时间限制
  | NoProgress         # 连续多轮无进展
  | LoopDetected       # 检测到循环模式
  | ConsecutiveFailures # 连续失败超限
  | TaskComplete       # 任务正常完成
  | Convergence        # 收敛（成功率过低）
```

**复杂度分级：**
| 复杂度 | 初始迭代 | 最大迭代 | 示例任务 |
|--------|----------|----------|----------|
| TRIVIAL | 3 | 5 | 打开应用 |
| SIMPLE | 5 | 15 | 列出文件 |
| MODERATE | 10 | 30 | 整理文件夹 |
| COMPLEX | 20 | 50 | 创建项目 |
| VERY_COMPLEX | 30 | 100 | 开发完整应用 |

**自愈流程：**
```
发现问题 → 诊断分析 → 生成修复计划 → 执行修复 → 验证结果 → 更新策略库
```

**支持的问题类型：**
- 工具解析失败 (`tool_parse_failure`)
- LLM 空响应 (`llm_empty_response`)
- Function Calling 失败 (`function_calling_failed`)
- LLM 连接错误 (`llm_connection_error`)
- 上下文丢失 (`context_lost`)
- 脚本执行错误 (`script_syntax_error`, `script_runtime_error`)
