# Chow Duck 官网

Chow Duck (MacAgent) 产品介绍官网。

## 开发

```bash
npm install
npm run dev
```

访问 http://localhost:5173

## 构建

```bash
npm run build
```

输出到 `dist/` 目录。

## 页面结构

- `/` — 首页
- `/features` — 功能特性
- `/technology` — 技术架构
- `/tools` — 工具落地
- `/skills` — 技能能力
- `/docs` — 文档索引
- `/help` — 使用帮助
- `/roadmap` — 待扩展 / 路线图
