# Mac Agent GUI Automation — 工业级优化清单（进度跟踪）

## 一、IPC 通信层

| 项目 | 状态 | 说明 |
|------|------|------|
| TCP 服务器（端口 8767） | ✅ 已完成 | IPCService.swift — NWListener TCP, 4-byte length prefix + JSON |

| ACK/NACK 消息类型 | ✅ 已完成 | IPCResponse type 支持 ack/nack |
| 多客户端/multi-agent 支持 | ✅ 已完成 | clientId 字段，per-connection 管理 |

## 关键文件清单

### Swift 侧（MacAgentApp）
- `Services/AccessibilityService.swift` — AXUIElement 封装
- `Services/AccessibilityBridge.swift` — HTTP Bridge 端口 5650
- `Services/AXObserverManager.swift` — AX 事件监听 + 崩溃恢复
- `Services/GUIStateStore.swift` — 版本化 GUI 世界模型
- `Services/ActionBatchExecutor.swift` — 12 种动作 + 批量执行
- `Services/IPCService.swift` — TCP + XPC 双传输 IPC 服务 + CRC32 校验 + 协议增强
- `Services/VisionFallbackService.swift` — 屏幕截图 + OCR + 模板匹配（Vision fallback）
- `Views/Monitor/AccessibilityMonitorView.swift` — A11Y HUD 监控面板

### Python 侧（backend）
- `runtime/ipc_client.py` — 异步 IPC 客户端
- `tools/dynamic_tool_generator.py` — GUIAutomationTool（IPC-first）
- `routes/permissions.py` — 权限检查 + IPC 状态
- `runtime/accessibility_bridge_client.py` — HTTP Bridge 客户端（fallback）
