## 二、核心架构变化

### 2.1 GUI 交互规范重构

**旧架构**：硬编码在 `autonomous_agent.py` 的 system prompt 中

**新架构**：`agent/shared_rules.py` 作为单一真相源

```python
# 动态注入到 prompt
{gui_rules}  # → 替换为 GUI_RULES 内容
```

**核心规则**：
- 禁止 `mouse_click` 点击已知元素 → 必须用 `gui_automation(click_element)`
- 禁止 `keyboard_type` 输入文字 → 必须用 `gui_automation(type_text)`
- 工作流：get_gui_state → find_elements → click_element → type_text → 截图验证

### 2.2 IPC 通信协议

**端口分配**：
| 端口 | 用途 | 协议 |
|------|------|------|
| 8765 | 主服务 HTTP | HTTP |
| 8766 | 主服务 WebSocket | WS |
| 8767 | Swift IPC | TCP + CRC32 |
| 5650 | Swift AX Bridge | HTTP |

**IPC 消息格式**：
```
[4字节长度][4字节CRC32][JSON payload]
```

**IPC 消息类型**：
- `ACTION` - 单动作执行
- `BATCH` - 批量执行
- `QUERY` - 世界状态查询
- `SUBSCRIBE` - 事件订阅
- `HEARTBEAT` - 心跳

### 2.3 Duck 分身架构
