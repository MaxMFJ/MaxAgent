# 主线目标与路线图

本文档合并「2026 标杆级 Agent」目标与「对标 Claude Code」进化，形成一条主线：当前状态 → 2026 标杆 → Claude Code 级体验 → 阶段里程碑。

---

## 一、目标定义

### 1.1 2026 标杆级 Agent 要件（六维度）

| 维度 | 标杆常见要件 | 典型参考 |
|------|----------------|----------|
| 架构与推理 | Plan-and-Execute、Replan、结构化/长程 memory、可选多智能体 | SWE-agent, Plan-and-Execute, Reflexion, MemoryAgentBench |
| 可观测与可评估 | 全链路 trace、成本/延迟/成功率可测、可复现 benchmark | MAESTRO, TRAIL, Traceloop, PacaBench |
| 安全与可控 | 沙箱、统一审计、危险动作拦截、HITL 审批/回滚 | CHEQ, Sherlock, GA-Rollback |
| 鲁棒与自愈 | 错误分类、智能 escalation、反思/验证、幂等 | v3 error_model, v3.1 escalation/reflection |
| 运维与扩展 | 深度健康检查、FeatureFlag、统一 ID | v3 部分已做 |
| 人机协同 | 关键动作确认、共规划、动作守卫、可回滚 | CHEQ, Magentic-UI, Sherlock |

### 1.2 Claude Code 级体验目标

| 维度 | Claude Code | MacAgent 当前 | 进化方向 |
|------|-------------|---------------|----------|
| **主循环** | 三阶段：gather → take action → verify，循环中自适应 | 单步 ReAct，无显式验证阶段 | 已加三阶段 prompt；代码层可选显式 Verify 阶段 |
| **上下文** | 自动压缩、CLAUDE.md 持久说明、/context 可查 | summarize_history + MACAGENT.md 注入 | 已达成项目级持久说明；可增 /context 类查询 |
| **会话** | Session 持久、resume/fork、编辑前 checkpoint | task_id/session_id，无 resume/fork/checkpoint | v3.2+ 可选：resume、checkpoint |
| **稳定性** | Extended Thinking、多模型、工具结果驱动 | 首步加固、trace、escalation（embedding） | Phase C：可选 CoT/Extended Thinking |
| **扩展** | Skills / Subagents / MCP / CLAUDE.md | Capsule、MCP 规划，无 Subagent | Phase C 可选 Subagent 设计 |

## 二、当前状态与差距

### 2.1 已具备（v3 + v3.1）

- **v3**：统一错误模型、任务状态机、并发限流、TimeoutPolicy 全链路、配置合并。
- **v3.1**：结构化 memory（`summarize_history_for_llm`）、Plan-and-Execute 可选、Escalation（embedding 相似度）、中途反思、危险命令/工具统一校验（`safety.py`）、Goal 重述、首步解析加固、MACAGENT.md 注入、轻量 trace（`data/traces/{task_id}.jsonl`）、三阶段 prompt（Gather/Act/Verify）、内部 benchmark 文档。

### 2.2 与 2026 标杆的差距

- **可观测**：trace 已落盘，待补执行轨迹持久化、token 统计与聚合。
- **可评估**：内部 benchmark 用例集已有，待自动化跑分与 CI 集成。
- **安全与运维**：沙箱、统一审计、`/health/deep` 仍为规划/占位。
- **人机协同**：HITL、回滚未做。

### 2.3 与 Claude Code 的差距

- 主循环仍为单步 ReAct，无代码级「验证」阶段。
- 无 resume/fork、无编辑前 checkpoint。
- 无 Extended Thinking / 长链推理。
- 无 Subagent。

## 四、路线图（v3.1 之后）

| 版本 | 内容 | 目标 |
